Linking through an image

This week I wanted to work on getting my About page set up for project1 that will include links to different social media like Twitter or Soundcloud. I think that the best way to achieve something like this is just having some simple icons for a user to click on. First, let's get some images in here.

Getting those icons on the page was easy enough, I just used some HTML that looks like this (styling done in a separate file of course):

<div class = "imgrow"> <img class = "logo" src = "img/twitter.png"> <img class = "logo" src = "img/soundcloud.png"> <img class = "logo" src = "img/reddit.png"> </div>

Now let's actually get these images functional. For demonstration purposes, I'll just have them link to the main page of their respective site.

How was this achieved you ask? Well, you could probably take a guess at the technique used...

<div class = "imgrow"> <a href = "https://twitter.com"> <img class = "logo" src = "img/twitter.png"> </a> <a href = "https://soundcloud.com"> <img class = "logo" src = "img/soundcloud.png"> </a> <a href = "https://reddit.com"> <img class = "logo" src = "img/reddit.png"> </a> </div>

Yup, that's right! Just a basic ol' <a> tag, like how you would link anything in HTML really. TLDR: I'm a dummy. Here's my source I used to learn about this. See ya next week!
